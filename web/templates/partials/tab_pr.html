• Translates press releases from labels/artists into Spanish and/or Brazilian Portuguese for LATAM distribution.
  • Paste the PR text directly or upload the .docx file from the label.
  • .docx mode preserves all formatting — fonts, bold, italic, sizes, images, headers, footers, and hyperlinks.
  • Artist names, song/album/EP titles, venue names, and URLs are kept as-is — only prose is translated.
  • Auto-detects the source language (English, French, Spanish, Portuguese).
  • Three engines: Gemini AI (primary), Mistral AI (fallback), and Google Translate (free). Each falls back to the next on failure.
  • Chunked API calls (15 paragraphs/batch) for reliability with large documents.
