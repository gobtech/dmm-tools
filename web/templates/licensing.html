About the Developer

LFP

Luis Felipe Pazos

@gobtech

The sole architect and developer behind DMM Tools. Luis Felipe designed and built this entire automation suite from the ground up — from the multi-platform DSP scraping engine and composite proof-image pipeline to the Soundcharts API integration, the 3-source press discovery system, and the full-stack web interface that ties it all together. What previously consumed dozens of hours of manual labor per week across the team now runs in seconds at the click of a button, thanks to his work.

Every line of code in this project — the backend orchestration, the API clients, the data processing pipelines, the report generators, the frontend UI — was conceived, written, and maintained by Luis Felipe. The system reflects a deep understanding of both the music marketing workflow it automates and the technical challenges involved in scraping, normalizing, and presenting data from six different streaming platforms, multiple search APIs, and a live release schedule with over 200 entries.

This is not a template or a fork. It is a purpose-built, production-grade toolchain engineered specifically for Dorado Music Marketing's operational needs, and it continues to evolve under Luis Felipe's direction.
