What is GANGSIGNS

GANGSIGNS is the proprietary search and intelligence engine that powers DMM Tools' Press Pickup system. It orchestrates a multi-source pipeline of free and paid search APIs, direct outlet integrations, and AI-powered filtering to find every press mention of an artist across Latin America — typically in under two minutes.

Unlike generic media monitoring services that rely on a single search provider, GANGSIGNS cross-references seven independent source layers against a curated database of 1,500+ LATAM media outlets, ensuring coverage that no single API can match alone. Results are deduplicated, date-verified, classified by outlet and country, and formatted into client-ready reports.

The Pipeline

GANGSIGNS executes its sources in sequence. Each layer fills coverage gaps left by the previous ones, and all results feed into a shared deduplication pool before classification.

1. RSS & WordPress Feed Scan
Scans 430+ outlet RSS feeds and WordPress REST APIs concurrently (50 workers). Dates are verified directly from feed metadata — no guesswork. Results are pre-classified with outlet name, country, and description from the feed registry.
Free · Unlimited
2. XML Sitemap Mining
Two-phase concurrent parse of XML sitemaps from 333 outlets that lack RSS feeds. Phase 1 fetches root sitemaps, Phase 2 drills into relevant monthly sub-sitemaps. For strict date windows, sitemap lastmod dates are re-verified against actual article HTML.
Free · Unlimited
3. Priority Outlet Adapters
Direct integrations with 7 high-priority LATAM outlets (Billboard Brazil, PopLine, Rolling Stone Mexico, El País Uruguay, RPP, BioBio Chile, Expreso Ecuador). Uses WordPress API or HTML scraping depending on the outlet.
Free · Unlimited
4. Google News RSS
Queries Google News across 9 LATAM regions (MX, AR, BR, CL, CO, PE, EC, UY, VE) with locale-tuned queries enriched from the release schedule. Decoded redirect URLs are resolved concurrently (50 workers). Failed decodes preserve source-name for downstream outlet matching.
Free · Unlimited
5. SearXNG Metasearch
Self-hosted metasearch engine (aggregates Google, Bing, DuckDuckGo) running in Docker. Searches both news and general web categories. Dates extracted from SearXNG metadata, URL patterns, and publishedDate fields.
Free · Self-hosted
6. Serper Targeted Search
Google SERP API with batched site: queries targeting unmatched outlets from the database. Groups up to 15 domains per query, localized by country majority. Fills coverage gaps from outlets that weren't found by free sources.
Paid · 3 credits/artist
7. Tavily & DuckDuckGo
Supplementary searches for additional coverage. Tavily runs news + general queries (free tier, 1,000/month). DuckDuckGo News provides one final sweep.
Tavily: Free tier
8. Free Targeted Retries
Final pass using Google News RSS with site: queries for remaining high-priority unmatched outlets. Batches 8 domains per query across 2 regions (MX + BR). Completely free — no API credits consumed.
Free · Unlimited

Intelligence Layer

Raw search results are meaningless without classification. GANGSIGNS applies multiple layers of processing to transform thousands of raw URLs into a structured, verified, and deduplicated press report.

Query Enrichment

Before any search is executed, GANGSIGNS loads the live release schedule from Google Sheets and builds context-aware queries. If an artist has a recent release, queries include the exact title and format keywords (nuevo sencillo, álbum, EP). For artists with ambiguous names (common words like "Future", "Metric", "Air", "Calle"), strict music-context shaping is applied to prevent false positives.

Date Verification

Every result carries a published_date and date_verified field. Dates are extracted from multiple signals: RSS pubDate, WordPress date_gmt, sitemap lastmod, SearXNG metadata, and — when needed — by fetching the article page itself and parsing HTML meta tags (article:published_time, og:published_time), <time> elements, JSON-LD datePublished, and URL date patterns.

For strict date windows (custom ranges or periods of 30 days or less), undated results are bulk-resolved by concurrently fetching article pages (10 workers), and sitemap dates are re-verified against actual article content. Unresolved results are dropped.

AI Relevance Filtering

Articles where the artist name only appears in the snippet (not the title) are likely false positives — sidebar mentions, related-article widgets, or substring collisions. GANGSIGNS sends these to Groq's Llama 3.3 70B model for title-only relevance judgement, with special handling for ambiguous artist names. Title-confirmed articles are auto-kept without AI review.

Outlet Classification

Results are matched against the 1,500+ outlet press database using domain matching (including subdomain resolution), Google News source-name fallback, and social media handle lookup via a curated registry of 480+ Instagram, Facebook, and X handles. Multi-regional outlet detection catches US editions of LATAM publications (generic .com articles from outlets that have country-specific domains in the database).

Deduplication

URLs are normalized (strip tracking parameters, /amp/, www., protocol differences) across all sources before dedup. Within outlet groups, title-based deduplication catches republished articles with different URLs using Unicode-normalized comparison.

Data Infrastructure

GANGSIGNS relies on several curated datasets, built and maintained through purpose-built discovery tools.

Press Database

1,500+ LATAM media outlets with name, country, description, and website URL. Originally sourced from Notion (226 with URLs), then enriched to 834 URLs via automated Brave Search discovery with confidence scoring.

Feed Registry

763 outlets scanned for RSS feeds and WordPress APIs. 424 RSS feeds + 6 WordPress APIs discovered, 333 outlets flagged for sitemap fallback. Generated by an automated feed discovery scanner that probes HTML link tags and common feed paths.

Social Handle Registry

763 outlets scanned for social media presence. 482 Instagram, 486 Facebook, and 429 X handles extracted from outlet homepages. Includes a reverse handle → outlet lookup for instant classification of social media posts found during search.

Release Schedule

Live Google Sheets integration with 220+ releases. Used for query enrichment (exact titles, format keywords) and batch mode operations (run all artists releasing this week).

Output

Results are grouped by outlet, then organized by country. Multiple articles from the same outlet are merged into a single entry with bulleted URLs. Article titles are used as hyperlink display text in reports.

Formats

  • Text report — Plain text with country headers, outlet names, descriptions, and URLs. Source breakdown footer shows how many results came from each pipeline stage.
  • Word document — Branded .docx with DMM styling (red accent headings, country sections, hyperlinked article titles). Ready for client delivery.
  • Google Docs — Auto-append to linked Google Docs for ongoing campaign tracking.

Batch Modes

  • Single artist — Full pipeline for one artist with custom date range
  • This week's releases — Sequential runs for all artists releasing this week, individual .docx files zipped
  • All releases — Full schedule batch with combined output
