Prototype Notice

  • This is an early-stage internal prototype, actively under heavy development
  • Not a polished or finished product — expect rough edges
  • Shared for internal demo and testing purposes only
  • Not intended for client-facing use or external distribution

What to Expect

  • Features may be incomplete, change significantly, or break between sessions
  • Data accuracy depends on upstream APIs and sources (Soundcharts, Google News, Spotify, etc.) — results should be verified
  • The UI is functional but not final — layouts, workflows, and visual design will continue to evolve
  • Performance may vary (some operations hit multiple external APIs sequentially)
  • No authentication or multi-user support — this is a single-user local tool
  • No automated test suite yet — manual testing is ongoing

Feedback & Issues

  • Bugs, suggestions, and feedback should go directly to Luis Felipe
  • Things will change between sessions as the tool evolves rapidly
  • If something breaks, report it — don't assume it's a known issue
